Flow for insert a unit in ingredients (an example):

After editing inventory/models.py (VS Code or CMD)

class Ingredient(models.Model):
#...
choices=[('kg', 'Kilograms'), ('g', 'Grams'), ('l', 'Liters'), ('unit', 'Units'), ('ml', 'Milliliters')],
#...

('ml', 'Milliliters') inserted

Run migrations in Docker

docker-compose exec web python manage.py makemigrations
docker-compose exec web python manage.py migrate

Rebuild dependencies:

docker-compose build
docker-compose up
